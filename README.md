# Fraud Detection System for NovaPay

## Project Overview
This project focuses on developing a machine learning-based fraud detection system for NovaPay, a rapidly growing digital money transfer company.

The company faces increasing challenges related to fraudulent financial activities, which threaten customer trust, financial security, and operational efficiency.

---

## Problem Statement
NovaPay experiences various fraudulent financial activities such as:

- Unauthorized transactions
- Identity theft
- Account takeovers
- Payment fraud

Traditional fraud detection systems are often unable to efficiently identify modern fraud patterns in real time.

Therefore, this project aims to develop an intelligent machine learning system capable of detecting fraudulent transactions automatically and accurately.

---

## Project Goal
The main goal of this project is to build a predictive model that can classify financial transactions into:

- Fraudulent Transactions
- Non-Fraudulent Transactions

The system is designed to help detect fraudulent activities in real time and improve transaction security.

---

## Target Variable
The target variable represents the fraud status of a transaction:

- `0` → Non-Fraudulent Transaction
- `1` → Fraudulent Transaction

---

## Machine Learning Model
This project uses Logistic Regression as the primary machine learning algorithm for binary classification.

The model is trained to learn transaction patterns and predict whether a transaction is fraudulent or legitimate.
